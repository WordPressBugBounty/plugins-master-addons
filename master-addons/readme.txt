== Builder Features ==

= Theme Builder =
Design every part of your site using dynamic templates with Elementor. Build and assign custom templates for:

- **Headers & Footers** ‚Äî Fully responsive, with sticky and transparent options
- **Single Post & Page** templates
- **Archive, Category, Tag, Author, Date** pages
- **404 Page** and **Search Results** page
- **Comment Forms** ‚Äî styled entirely in Elementor
- Display conditions let you assign each template to specific pages, post types, or user roles

[Learn More about Theme Builder ‚Üí](https://master-addons.com/theme-builder/?utm_source=wordpressorg&utm_medium=readme&utm_campaign=master-addons)

= Popup Builder =
Create professional popups directly in Elementor ‚Äî no third-party plugin needed. Choose from 5 smart triggers to control exactly when a popup fires:

- **Page Load** ‚Äî with configurable delay
- **On Scroll** ‚Äî appears after user scrolls 25%, 50%, 70%, or 90% down the page
- **Exit Intent** ‚Äî captures visitors before they leave
- **On Click** ‚Äî fires only when a user clicks a specific button or link
- **After Inactivity** ‚Äî re-engages idle visitors automatically

Target your popup to any page, post type, or section using built-in Display Conditions.

[Learn More about Popup Builder ‚Üí](https://master-addons.com/popup-builder/?utm_source=wordpressorg&utm_medium=readme&utm_campaign=master-addons)

= Widget Builder =
Turn any custom HTML, CSS, or JavaScript ‚Äî including AI-generated code or CodePen snippets ‚Äî into a reusable, fully editable Elementor widget. No PHP required:

- Add controls (text, color, toggles, select dropdowns) from a visual panel
- Auto-generated shortcodes like `{{color}}` map live editor inputs directly to your code
- Save once and drag the widget onto any page like a native Elementor element

[Learn More about Widget Builder ‚Üí](https://master-addons.com/widget-builder/?utm_source=wordpressorg&utm_medium=readme&utm_campaign=master-addons)

= Template Kits & Template Library =
Launch complete websites in minutes. Master Addons ships with a growing collection of ready-to-use Template Kits ‚Äî full site designs with coordinated pages, headers, footers, and inner pages. Import any kit with one click straight from inside your WordPress dashboard.

[Browse Template Kits ‚Üí](https://master-addons.com/template-kits/?utm_source=wordpressorg&utm_medium=readme&utm_campaign=master-addons)

= Mega Menu Builder =
Build feature-rich mega menus for your site directly in Elementor. Use any Elementor widget inside the menu panel, add 3,000+ icons, configure hover and click events, and style every detail with full color and typography controls.

[Learn More about Mega Menu ‚Üí](https://master-addons.com/extension/mega-menu/?utm_source=wordpressorg&utm_medium=readme&utm_campaign=master-addons)

== Frequently Asked Questions ==

= Can I use Master Addons without the Elementor plugin? =
No. Elementor page builder is required. If it is not installed, Master Addons will display an installation notice automatically.

= How do I install Master Addons? =
In your WordPress admin go to **Plugins ‚Üí Add New**, search "Master Addons for Elementor" and click Install Now. Alternatively, upload the ZIP via FTP. See video tutorials on our [YouTube Channel](https://www.youtube.com/channel/UCAPfTXvzbNebKsB322Iz6HQ).

= Do I need to configure anything after activation? =
No configuration is required. Head to the Master Addons panel to enable or disable individual widgets and extensions to keep your site loading fast.

= How do I add a Master Addons widget to a page? =
Open Elementor on any page or post. Search "Master Addons" in the widget panel or scroll to the Master Addons category. Drag any widget onto your canvas and customize it.

= Is Master Addons compatible with all WordPress themes? =
Yes. It works with any theme that supports Elementor.

= How do I fix "Master Addons installed but not working"? =
First, deactivate other addon plugins and try again. If the issue persists, post a ticket on the [WordPress Support Forum](https://wordpress.org/support/plugin/master-addons/) or reach us via live chat.

= Where do I report security vulnerabilities? =
Report security issues through the [Patchstack Vulnerability Disclosure Program](https://patchstack.com/database/vdp/9e5fb6e3-e633-4f5e-853f-21e88c02fd30). The Patchstack team handles verification, CVE assignment, and developer notification.
